Starting the Application

Before starting the application, ensure that the System Requirements have been met and that the application is installed and configured correctly.

The Java Archive file can be executed in one of three modes:

  • The Data Client in Networked Mode (by not passing any command line argument)
    • > java -jar runme.jar

  • The Data Server (by passing the command line argument "server")
    • > java -jar runme.jar server
  • The Data Client in Local Mode (by passing the command line argument "alone")
    • > java -jar runme.jar alone

The Data Server

Before starting the data server, make sure it is configured correctly. Once the application has been started in server mode, it will only respond to simple commands, such as exiting the application. There is no ability to stop the server after it has been started, except for exiting the application entirely. If the server has a problem starting or a problem during operation, an exception will be reported to the server dialog application.

The Data Client

Once the data client has started, a window will be displayed that contains various buttons and menu options. If the client has been configured correctly, click the connect button to connect to the data source.
